# Web Journey: Закрывающий тег

Уникальный авторский блог, посвященный пути в веб-разработке. Проект выполнен в стиле **Cyber-Minimalism** с использованием современных технологий верстки.

## Особенности проекта

- **Уникальный дизайн**: Полностью переработанный интерфейс с эффектами Glassmorphism и плавной типографикой.
- **Адаптивность**: Идеальное отображение на любых устройствах (от смартфонов до 4K мониторов).
- **Темная тема**: Поддержка системных настроек и ручного переключения темы.
- **Анимации**: Плавные entrance-анимации и микро-взаимодействия.

## Технологии

- **HTML5**: Семантическая верстка.
- **SCSS**: Современная модульная архитектура стилей.
- **CSS Grid & Flexbox**: Сложные и гибкие сетки.
- **JavaScript**: Управление темами и интерактивность.
- **Gulp**: Автоматизация сборки и оптимизация ресурсов.
