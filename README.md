# Web Journey: Закрывающий тег

Уникальный авторский блог, посвященный пути в веб-разработке. Проект выполнен в современном минималистичном стиле с поддержкой темной темы.

## Особенности проекта

- **Чистый дизайн**: Современный интерфейс с акцентом на контент и типографику.
- **Адаптивность**: Идеальное отображение на любых устройствах.
- **Темная тема**: Удобное переключение между светлым и темным режимами.
- **Микро-взаимодействия**: Плавные анимации и интерактивные элементы.

## Технологии

- **HTML5**: Семантическая верстка.
- **CSS3**: Современные стили и Flexbox/Grid.
- **JavaScript**: Управление темами и логика лайков.
